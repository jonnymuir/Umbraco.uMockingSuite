## Project Context

- **Owner:** Jonny Muir
- **Project:** uMockingSuite — A Umbraco 17 backoffice package. When a content editor creates or updates content in the Umbraco backoffice, the package intercepts the event and displays a witty, mocking notification or advisory message. Think passive-aggressive spell-checker energy, but ruder.
- **Stack:** C# / .NET 9, Umbraco 17, TypeScript, Lit (Umbraco backoffice UI), Umbraco Package Manifests
